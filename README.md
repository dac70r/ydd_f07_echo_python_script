# ydd_f07_echo_python_script
Python script to automate sending mouse and keyboard packets to ps2 emulator/ controller. 

To run this code: 

1. Download relevant python packages
2. Check the COM port of the PS2 Controller using a terminal software
3. To test Keyboard: type python auto_test_keyborad_script.py in terminal
4. To test Mouse: type python auto_test_mouse_script.py in terminal
